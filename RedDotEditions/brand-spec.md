# red dot editions — brand spec

Light field, crimson voice, Karasuma Gothic type, solid floating dots.

## Tokens

```css
:root {
  --bg: #f0f0f0;
  --surface: #ffffff;
  --fg: #d6001a;
  --muted: color-mix(in srgb, #d6001a 58%, #f0f0f0 42%);
  --border: color-mix(in srgb, #d6001a 28%, transparent);
  --accent: #d6001a;
  --on-accent: #f0f0f0;
}
```

## Type

- **Karasuma Gothic** — body Regular/Medium; brand name always **Bold** and **lowercase**: `red dot editions`
- Local: `assets/fonts/KarasumaGothic-*.woff2`

## Posture

1. Canvas `#f0f0f0`, one crimson `#d6001a`.
2. No section index numerals; titles lead the two-column grid.
3. Brand name never title case / all caps.
4. Solid dots only (no arrows).
5. Pages: `index.html` (PT), `index-en.html` (EN), `faq.html` / `faq-en.html`, `quem-somos.html` / `about.html`.
